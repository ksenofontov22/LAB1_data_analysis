# Лабораторная работа №1: Первичное исследование и оценка качества данных

## Описание проекта
Первичный анализ датасета "hourly_weather_data" в рамках дисциплины "Интеллектуальный анализ данных".

**Предметная область:** Датасет "hourly_weаther_data" содержит данные о метеорологических и атмосферных условиях, собираемых датчиками с переодичностью в 1 час. Сбор данных производился с 11 января 2011 года по 15 февраля 2014 года. Возможными задачами ИАД являются категоризация погодных условий местности, сбор статистики по частоте погодных явлений.
**Цель лабораторной:** аудит качества данных, построение паспорта датасета, выявление проблем.

## Структура проекта
- `notebooks/01_data_understanding.ipynb` — основной Jupyter Notebook с анализом  
- `report/quality_report.md` — текстовый отчёт с выводами и планом улучшения  
- `data/` — датасет или скрипт загрузки
